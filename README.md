# React Mini Projects

A collection of small React projects created to practice and demonstrate core React concepts.

## Purpose

This repository is built to strengthen React fundamentals through focused mini projects.  
Each project targets specific concepts commonly expected at a junior frontend developer level.

## Covered Concepts

- Component-based architecture
- Props and data flow
- useState for state management
- Event handling (onClick, onChange)
- Rendering lists with map and key
- Reusable components
